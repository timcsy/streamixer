# API 契約：即時跳轉與背景預生成

## 端點變更

本功能不新增或修改任何外部 API 端點。既有的三個端點行為不變：

- `GET /stream/{id}/index.m3u8` — 回傳 HLS playlist（內部改為觸發背景預生成）
- `GET /stream/{id}/seg_{n}.ts` — 回傳分段（內部改為優先使用預生成分段）
- `GET /health` — 不變

## 行為變更（對使用者透明）

### GET /stream/{id}/index.m3u8

**新增行為**：首次請求時，在回傳 playlist 的同時背景啟動預生成任務。
**使用者感知**：無變化，回應時間不受影響。

### GET /stream/{id}/seg_{n}.ts

**新增行為**：
1. 優先檢查預生成的分段是否存在
2. 若存在 → 直接回傳（零合成延遲）
3. 若不存在 → 按需生成（與目前行為相同，但修正時間戳對齊）

**使用者感知**：分段回應速度提升，跳轉後音訊對齊。
